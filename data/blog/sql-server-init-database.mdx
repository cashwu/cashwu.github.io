## 問題

在 windows 的事件裡面一直出現 `正在啟動資料庫 Adventure Works2012`

![](/static/images/404.webp)

## 解法

```sql
ALTER DATABASE AdventureWorks2012
SET AUTO_CLOSE OFF
```

![](/static/images/404.webp)
