## 問題

使用 Code First 產生 DB 後，要建立資料庫圖表時會出現訊息說沒有 `有效的擁有者`

![](https://image.cashwu.com/404.webp)

## 解法

- DB 上面按右鍵選擇 `屬性`

![](https://image.cashwu.com/404.webp)

- 在 `檔案` 裡面的 `擁有者` 選擇一個 DB 的登入者

![](https://image.cashwu.com/404.webp)

- 再一次建立時就可以了

![](https://image.cashwu.com/404.webp)
