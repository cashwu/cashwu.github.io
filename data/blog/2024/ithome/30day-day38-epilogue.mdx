終於結束了這次的鐵人賽，最後除了前後的兩篇~廢文~之外，想不到總共寫了 36 篇文章

其實早在好幾年前就有想要參加

---

一開始想寫的題目是 NET Core 的 End to End 測試，當時的版本應該是 2.2，剛好出了這個神奇的東西

我就研究了一段時間，也有實作在專案上面，也有在公司內部，作了一次相關的教育訓練

不過後來還是沒有報名

---

最近的一次，是有關於 NET Core 和 GCP 的相關

那個時候，公司的專案只可以使用 GCP 的相關服務，而當時在網路上也不好找到 NET Core 和 GCP 的相關資料

自己也花了蠻多時間在摸索的，本來想把這些經驗分享出來

不過後來也是沒有報名

---

今年，也是個神奇的機會，開始學習 Spring Boot

在網路上找到的實作版本不一，舊版蠻多的東西都已經棄用了，有些有新版本的實作，但卻不是那麼的完整

自己也踩了不少的坑，想說這也是一個從 0 到 1 的過程

在鐵人賽開賽了半個月後，終於下定決心

報名了鐵人賽「Spring Boot API 開發：從 0 到 1」

---

一開始的主題大小和分配，真的不好拿捏，有些拿掉又覺得可惜，有些又覺得要多一點

一邊寫一邊調整，來來回回的情況下，最後終於定案，寫了 36 篇文章

有些文章的主題/內容，其實沒有講的很細，有些又覺得講的太細了，總之就是一個取捨

以 Bean 為例，真的每個細節都要說的話，應該可以寫好幾篇文章了

秉持著一個「其實不用懂這麼多，就可以開發 Spring Boot 的 API 的信念」，寫了這些文章

---

這整個過程中，雖然很累人，但也是收穫不少

最後有完成，就是一件很棒的事情，有沒有得獎，真的就是其次

希望這一系列的文章，能夠幫助到有需要的人

也希望明年有好的題目，可以再參加鐵人賽，寫更多文章，分享更多知識
