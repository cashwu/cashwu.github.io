## 問題

專案在發佈到 Azure 的 WebSite 後，執行時發生找不到某個 View

## 原因

- 在專案的設定檔 (`.csproj` ) 內，找不到的那個 View 為 `None`

![](/static/images/404.webp)

## 解法

在專案檔裡面把 `None` 改為 `Content` 就好了
