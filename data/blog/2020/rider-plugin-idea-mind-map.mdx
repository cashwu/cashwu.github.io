> 套件 [IDEA Mind Map](https://plugins.jetbrains.com/plugin/8045-idea-mind-map)

> 這個套件是從 [NetBeans 的 MindMap](http://www.igormaznitsa.com/netbeans-mmd-plugin/) 移植而來

## 新檔案

安裝完套件後，在任一專案裡面的 `新增檔案` 裡面就會出現 `Empty Mind Map` 的檔案

![](/static/images/2020/rider-plugin-idea-mind-map/01.webp)

選擇後，給定檔案名稱

![](/static/images/2020/rider-plugin-idea-mind-map/02.webp)

就會進入 `Mind Map` 的編輯畫面

![](/static/images/2020/rider-plugin-idea-mind-map/03.webp)

這個檔案就可以加入版控，團隊共享

![](/static/images/2020/rider-plugin-idea-mind-map/04.webp)

實際上的 `mmd` 檔案也是 `Markdown`，如果用文字編輯器開可以看到裡面的格式大制上是 `Markdown`

![](/static/images/2020/rider-plugin-idea-mind-map/05.webp)

## 子節點

在節點上面按 `右鍵` 就會出現選單，選擇 `Add child` 會可以加一個子節點 (按 `Enter` 可以快速建立子節點)

![](/static/images/2020/rider-plugin-idea-mind-map/06.webp)

![](/static/images/2020/rider-plugin-idea-mind-map/07.webp)

## 編輯文字

在節點上按 `兩下` 就可以編輯文字

![](/static/images/2020/rider-plugin-idea-mind-map/08.webp)

## 物件

在節點上面可以新增不同的物件

-   note
-   file
-   URI
-   jump
-   Image
-   Emoticons

![](/static/images/2020/rider-plugin-idea-mind-map/09.webp)

新增之後選項會變為 `Edit`

![](/static/images/2020/rider-plugin-idea-mind-map/10.webp)

## Import

可以從不同的 MindMap 軟體 import 進來，一些主流的軟體都有

![](/static/images/2020/rider-plugin-idea-mind-map/24.webp)

## Export

相對的，也可以 export 到不同的 MindMap 軟體

![](/static/images/2020/rider-plugin-idea-mind-map/25.webp)

要注意的是，export 為 `Markdown` 的話，加入的檔案和圖會變成 `檔名`，下面為對造圖
